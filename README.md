# Claude Code Plugins

A collection of [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugins. I create and share plugins to automate processes and delegate real-work to the AI that gets stuff done.

## Plugins

| Plugin                                           | Description                                                                             |
| ------------------------------------------------ | --------------------------------------------------------------------------------------- |
| [feynman-course-plugin](./feynman-course-plugin) | Transforms complex topics into structured, first-principles courses published to Notion |

## Usage

Add any plugin to your Claude Code session:

```bash
claude --plugin-dir ./<plugin-name>
```
